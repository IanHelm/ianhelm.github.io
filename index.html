Sample Projects

Mapping Out Crime in Washington D.C.

This web mapping application was my final project for GGS 462 (Web Mapping). This involved the use of layer functionalities such as dropdown boxes and click functionality to create an interactive map of crime incidents from 2017. Users can narrow crimes by offenses and click on points to display date of occurrence. The application retrieves crime data dynamically from a CartoDB database using an SQL API call.

Boom or Bust: Recreation vs. Mining in Utah

This group research project for GGS 463 (Applied Geographic Information Systems) was done by me, Yolani Martin, Adam Black, and Steven Alvarez. The objective was to test our hypothesis that tourism had a stronger effect on Utah’s economy as opposed to mining activities. This was done through producing reference maps, Getis-Ord and Multiple Regression analysis. The finished project was presented at GIS Day 2018 at George Mason University.

View Project

Correlating Household Poverty and Graduation Rates in Baltimore, MD

This was my final project for GGS 310 (Intro to Digital Cartography). This involved creating a map that displayed the correlation between household poverty and graduation rates in each neighborhood of Baltimore City, Maryland. A scatterplot was also created in the project, which demonstrated a slight negative correlation between the two variables.

View Project

Hot Spot Analysis of Obesity in Illinois

This side project was made by conducting a Getis-Ord hotspot analysis of obesity rates in each county of Illinois. Based on the results that were produced, there were major cold spots around the Chicago Metropolitan Area, indicating low rates of obesity in that region.
